Northwind Step by Step Installations Guide:

Installations:

sudo apt update
sudo apt upgrade
sudo apt install nodejs
sudo apt install npm

sudo npm install -g create-react-app
sudo npm install -g json-server

After this installations, I must create a project folder. In Desktop:

sudo mkdir northwind-redux

then in the northwind-redux folder you must do following installations:

npm install redux
npm install react-redux
npm install bootstrap
npm install reactstrap
npm install redux-thunk
npm install alertifyjs
npm install react-router-dom
